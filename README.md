# Password-Manager
Web based Password Manager App 👨‍💼

## Features 🚀
- Complete User Authentication with **Google Auth** 🔐
- Store / Update / Delete / Get Passwords 🔑
- **Password Strength Checker** 💪
- **Generate Passwords** 📰
- **Add Notes** 📒
- Share Passwords with unique **QR Code** 🔗

## Tech Stack 🥪
**Backend**
 - Express.js 🚇
 - MongoDB 🍃
<br>

**Frontend**
 - React.js 💫
 - SASS 🟣
 - React Router 🎯
 - React Redux 🏪
